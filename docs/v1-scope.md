# V1 Scope

BrainBridge V1 focuses on making vaults understandable outside Obsidian.

## Included

- Read-only vault scanning.
- Markdown wikilink and embed extraction.
- Local Markdown link and image extraction.
- Broken link and missing attachment detection.
- YAML frontmatter property indexing.
- Markdown checkbox task indexing.
- Obsidian block reference inventory on links.
- `.obsidian` configuration inventory.
- Community plugin inventory.
- Canvas and Bases detection.
- Dataview, Tasks, Templater, Kanban, Excalidraw, and Obsidian URI detection.
- Graph, backlinks, attachment index, task index, property index, plugin dependency report, Markdown fallback, Canvas fallback, Bases fallback, and degradation report generation.
- Portable export bundle.

## Not Included

- Source vault mutation.
- Full Dataview or Tasks query evaluation.
- Templater execution.
- Full Bases formula evaluation.
- Full Excalidraw rendering.
- Live Canvas layout rendering.
- Obsidian plugin runtime compatibility.
- Sync service migration.
