What BrainBridge preserves

Plain files for readers, indexers, static sites, and agents that do not run Obsidian.

Structure

Backlinks, graph edges, attachments, note properties, and task lists.

Degradations

Plugin dependencies, missing embeds, broken links, and external state.

Fallbacks

Readable Markdown for Dataview, Tasks, Canvas, Bases, and Excalidraw notes.
